## LV-Demo

Louis Vuitton Demo를 위한 애플리케이션 입니다.

### Features

- 360 rotate viewer : 수평 방향 스와이핑, 감/가속에 대한 스와이핑 속도 조절 가능
- 각도 이미지별 zoom, pan 지원

### Features

- react-swipeable
- react-zoom-pan-pinch
- styled-component
